# Python 简介
哈哈

**Python** 是一门简洁、易读、功能强大的通用编程语言，广泛应用于数据分析、人工智能、Web 开发、自动化脚本等领域。

## 为什么选择 Python？

- **语法简洁**：接近自然语言，适合初学者快速上手
- **生态丰富**：拥有海量第三方库（NumPy、Pandas、Flask、Django……）
- **跨平台**：Windows、macOS、Linux 均可运行
- **社区活跃**：全球最大的编程社区之一

## 安装 Python

前往 [python.org](https://www.python.org/downloads/) 下载并安装最新版本。

安装完成后，在终端运行：

```bash
python --version
```

## 基本语法

```python
# 变量与类型
name = "HaoYe"
age = 18
pi = 3.14

# 列表
fruits = ["苹果", "香蕉", "橙子"]

# 字典
person = {"name": "HaoYe", "age": 18}

# 条件判断
if age >= 18:
    print("成年人")
else:
    print("未成年")

# 循环
for fruit in fruits:
    print(fruit)

# 函数定义
def greet(name):
    return f"你好，{name}！"

print(greet("Python"))
```
