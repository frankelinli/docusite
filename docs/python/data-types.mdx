# 数据类型

Python 内置多种数据类型，掌握它们是编程的基础。

## 数值类型

```python
# 整数
x = 42

# 浮点数
y = 3.14

# 复数
z = 2 + 3j
```

## 字符串

```python
s = "Hello, Python!"

# 常用方法
print(s.upper())       # 转大写
print(s.lower())       # 转小写
print(s.replace("Python", "World"))  # 替换
print(len(s))          # 长度

# f-string 格式化
name = "HaoYe"
print(f"我叫 {name}，今年 {2026 - 2000} 岁")
```

## 列表（List）

```python
nums = [1, 2, 3, 4, 5]

nums.append(6)       # 添加元素
nums.remove(1)       # 删除元素
nums.sort()          # 排序
print(nums[0])       # 索引访问
print(nums[-1])      # 最后一个元素
print(nums[1:3])     # 切片
```

## 字典（Dict）

```python
info = {
    "name": "HaoYe",
    "age": 18,
    "skills": ["Python", "VBA", "JavaScript"]
}

print(info["name"])
info["city"] = "Beijing"   # 添加键值对
del info["age"]            # 删除键

for key, value in info.items():
    print(f"{key}: {value}")
```

## 元组与集合

```python
# 元组（不可变）
point = (10, 20)

# 集合（去重）
tags = {"Python", "AI", "Python", "数据"}
print(tags)  # {'Python', 'AI', '数据'}
```
